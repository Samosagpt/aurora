# Samosa GPT Portable

This is a portable version of Samosa GPT that runs with your system Python.

## Setup
1. Run setup.bat to install dependencies
2. Wait for installation to complete

## Running
- Web version: Double-click run_web.bat
- Console version: Double-click run_console.bat

## Requirements
- Python 3.8+ must be installed on your system
- Internet connection for initial setup

## First Run
The first time you run the application, it may take longer as it downloads AI models.

## Troubleshooting
- If setup.bat fails, try running: pip install -r requirements.txt
- For audio issues, ensure microphone permissions are granted
- For web version, open http://localhost:8501 in your browser
